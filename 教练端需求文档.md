# 教练端需求文档

## 项目概述

教练端是一个专为网球教练设计的移动端应用，用于管理课程预约、学员信息、场地安排等教学相关业务。

## 功能模块

### 1. 首页 (Home)
- 显示教练的基本信息和概览数据
- 快速导航到各个功能模块

### 2. 课程管理 (我的课程)
- **课程列表展示**
  - 显示所有课程预约信息
  - 支持按状态筛选：全部、待确认、已确认、已完成
  - 课程卡片包含：课程名称、学员姓名、时间、地点、状态、价格、备注

- **课程状态管理**
  - 待确认 (pending)：新的预约请求
  - 已确认 (confirmed)：已确认的课程
  - 已完成 (completed)：完成的课程
  - 已取消 (cancelled)：取消的课程
  - 已更新 (updated)：信息有更新的课程
  - 已结束 (ended)：结束的课程，显示"查看评价"链接

- **课程详情**
  - 详细的课程信息展示
  - 学员联系方式
  - 课程备注信息

### 3. 订场管理 (订场)
- 场地预订信息管理
- 空白页面（当前无内容）

### 4. 学员管理 (Students)
- 学员信息管理
- 学员课程历史记录
- 学员联系方式

### 5. 个人资料 (Profile)
- 教练个人信息管理
- 账户设置
- 系统设置

### 6. 课程创建 (CreateCourse)
- 创建新的课程类型
- 设置课程参数（时长、价格、人数等）

### 7. 时间安排 (TimeSlotSetting)
- 教练可用时间段设置
- 时间表管理

## 用户界面设计

### 导航结构
- 底部导航栏包含主要功能模块
- 页面间使用React Router进行路由管理

### 卡片设计
- 统一的卡片样式用于展示课程信息
- 状态颜色编码：
  - 待确认：警告色 (warning)
  - 已确认：主色调 (primary)
  - 已完成：成功色 (success)
  - 已取消：危险色 (danger)
  - 已更新：蓝色 (#007AFF)
  - 已结束：静音色 (muted)

### 交互特性
- 点击卡片查看详情
- 已结束课程显示"查看评价"链接
- 状态筛选功能
- 响应式设计适配移动端

## 数据模型

### 课程预约 (Booking)
```typescript
interface Booking {
  id: number
  courseName: string        // 课程名称
  studentName: string       // 学员姓名
  studentAvatar?: string    // 学员头像
  date: string             // 日期
  time: string             // 时间段
  location: string         // 场地位置
  storeName: string        // 门店名称
  storeAddress: string     // 门店地址
  status: 'pending' | 'confirmed' | 'completed' | 'cancelled' | 'updated' | 'ended'
  price: number            // 价格
  note?: string            // 备注
}
```

## 业务流程

### 课程预约流程
1. 学员发起课程预约请求
2. 教练收到待确认的预约 (pending)
3. 教练确认预约，状态变为已确认 (confirmed)
4. 课程进行中，状态可能更新为已更新 (updated)
5. 课程完成后，状态变为已结束 (ended)
6. 已结束的课程可查看学员评价

### 课程管理流程
- 教练可以查看所有状态的课程
- 支持按状态筛选课程列表
- 可以查看课程详细信息
- 已结束课程提供评价查看功能
